# WezTerm Config

## 外観

| 設定 | 値 |
|---|---|
| カラースキーム | Pop Candy |
| フォント | Cica (Medium) / 14pt |
| 背景透明度 | 0.75 |
| ペイン境界線 | `#FF6B6B` (赤系アクセント) |

## Leader キー

**`Ctrl + B`** (1秒以内に次のキーを押す)

tmux と同じプレフィックス方式。

## キーバインド一覧

### ペイン操作 (Leader + key)

| キー | 動作 |
|---|---|
| `Leader` + `v` | 左右に分割 |
| `Leader` + `s` | 上下に分割 |
| `Leader` + `z` | ペインをズーム / 解除 |
| `Leader` + `x` | ペインを閉じる (確認あり) |
| `Leader` + `h/j/k/l` | ペイン移動 |

### タブ操作

| キー | 動作 |
|---|---|
| `Leader` + `c` | 新規タブ |
| `Ctrl` + `t` | 新規タブ |
| `Ctrl` + `w` | タブを閉じる (確認あり) |
| `Ctrl` + `Tab` | 次のタブ |
| `Ctrl` + `Shift` + `Tab` | 前のタブ |

### ペイン移動 (Leader なし)

| キー | 動作 |
|---|---|
| `Ctrl` + `h/j/k/l` | ペイン移動 (vim/nvim 内ではウィンドウ移動に透過) |

### 編集補助

| キー | 動作 |
|---|---|
| `Ctrl` + `←/→` | 単語単位で移動 |
| `Ctrl` + `Backspace` | 単語削除 |

## モード

モード中はタブバー右上にモード名と操作キーが表示される。
`Esc` または `q` でモードを抜ける。

### RESIZE モード (`Leader` + `r`)

ペインサイズを調整する。

| キー | 動作 |
|---|---|
| `h` | 左に広げる |
| `j` | 下に広げる |
| `k` | 上に広げる |
| `l` | 右に広げる |

※ 10px 単位で調整。連打可。

### OPACITY モード (`Leader` + `o`)

背景の透明度を調整する。

| キー | 動作 |
|---|---|
| `k` | 透明度を上げる (不透明に) |
| `j` | 透明度を下げる (透明に) |

※ 15% 単位で調整。範囲: 10% ~ 100%。

### FONT モード (`Leader` + `f`)

フォントサイズを調整する。

| キー | 動作 |
|---|---|
| `k` | 大きくする (+1pt) |
| `j` | 小さくする (-1pt) |
| `0` | デフォルト (14pt) にリセット |

※ 最小 6pt。

### COPY モード (`Leader` + `[`)

vim 風のテキスト選択・コピー。WezTerm 組み込み機能。

| キー | 動作 |
|---|---|
| `h/j/k/l` | カーソル移動 |
| `w/b` | 単語単位で移動 |
| `0/$` | 行頭 / 行末 |
| `g/G` | バッファ先頭 / 末尾 |
| `v` | 選択開始 (Visual) |
| `V` | 行選択 |
| `y` | コピーしてモード終了 |
| `/` | 検索モードに入る |

## デフォルトシェル

|---|---|
| Windows | PowerShell 7 |
| Linux | zsh (なければ bash) |
| macOS | zsh |
